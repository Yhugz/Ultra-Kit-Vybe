## POUR CHAQUE PAGE DU SITEMAP

Génère dans l'ordre :

1. HERO
   - H1 (6 mots maximum)
   - Sous-titre (15 mots maximum)
   - CTA principal (3 mots maximum)

2. SECTIONS PRINCIPALES
   - Titre de section H2
   - 3 blocs bénéfices (titre + description 2-3 lignes)

3. PREUVE SOCIALE
   - Structure de témoignage type
   - Indicateurs d'autorité (chiffres, labels, certifications)

4. CTA SECONDAIRES
   - 3 variantes de call-to-action selon l'intention

5. SEO
   - Meta title (60 caractères max)
   - Meta description (155 caractères max)
   - Balises alt pour les images principales

6. FOOTER
   - Navigation structurée
   - Mentions légales type
   - Accroche sociale

## CONTRAINTES

- Aucun anglicisme sauf si usage professionnel accepté
- Phrases courtes, actives, directes
- Intégrer les mots-clés SEO naturellement
- Préciser la hiérarchie (H1 / H2 / H3 / Body / Caption)
- Préciser le nombre de caractères pour chaque bloc

# Prompt 1 — Architecture

Tu es un architecte web senior. Je dois créer un site Next.js 14 pour le client suivant :

CLIENT
- Nom : [NOM]
- Secteur : [SECTEUR]
- Type de site : [vitrine / e-commerce / portfolio / SaaS]

AUDIENCE
- Cible principale : [DÉCRIRE]
- Objectif principal du visiteur : [CONVERTIR / INFORMER / ACHETER]

PRIORITÉS TECHNIQUES
- [SEO / PERFORMANCE / RESPONSIVE / ANIMATIONS]

Génère :

1. SITEMAP COMPLET
   - Toutes les pages avec leur hiérarchie
   - URL slugs propres en français

2. PARCOURS UTILISATEURS
   - 3 chemins de conversion principaux (entrée → action)

3. INVENTAIRE DES SECTIONS
   - Pour chaque page, liste toutes les sections dans l'ordre

4. COMPOSANTS NÉCESSAIRES
   - Liste tous les composants UI réutilisables à créer

5. INTÉGRATIONS REQUISES
   - Formulaires, APIs, outils tiers nécessaires

Format : structure claire, prête à être utilisée comme base de développement dans Cursor.
